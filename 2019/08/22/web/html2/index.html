CSS简介

        CSS 指层叠样式表(Cascading Style sheets)
        定义样式如何显示HTML。更好的渲染html页面，让它看上去更加的绚丽多彩。
        有3种展示展示方式：
        1.外部样式
            定义在以.css为后缀的文件当中，然后在.html为后缀的文件中引入。
            方便维护。提高工作效率
        2.内部样式
            写在html文件中的head标记中。
        3.内联样式
            写在html标签中。
        通常css样式可以写在.css文件当中，也可以在HTML文件当中，使用<style>标签来声明。
